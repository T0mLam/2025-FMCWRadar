# User Stories

Understanding user stories and their purpose

## Car Owner
- As a *car owner*, I want to be able to open my car door hands-free,  
  so that I don’t have to put down my shopping, child, or anything else in my hands.  
  It also prevents my hands from getting wet from a wet handle.

## Security Personnel
- As *security personnel*, I want there to be an early warning system for people being detected,  
  so that if I am not actively monitoring surveillance, there is an extra layer of security.
